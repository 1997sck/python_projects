                                                             Voice Recorder
                                                         
                                                  Install a packages by using following command
                                                  
                                                     pip3 install sounddevice
                                                     
                                                     To install wavio:
                                                     
                                                     pip3 install wavio
                                                     
                                                     To install scipy:
                                                     
                                                     pip3 install scipy
                                                     
                                                     Run the program Voice_recorder.py
                                                     
                                                Voice will be recorded and saved file as recording1.wav in the project directory
